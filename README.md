## Orchestrate Jenkins Master/Slave with Docker Compose

### Requirement (Ubuntu user)
1. Docker
2. Docker Compose

### Requirement (Mac/Windows user)
1. virtualbox
2. vagrant

### Get source
    git clone https://github.com/moiamond/jenkins_study
    git submodule update --init --recursive
  
### Linux user
    docker-compose up
  
### Windows/Mac user
    vagrant up
    
### Have fun
    http://localhost:8080
